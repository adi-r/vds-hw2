Navigating the White Hat Viz.

  • The visualiztion is made by integrating choropleth and bubble maps for state level and city level statistics respectively.
    The color scheme used is color blind safe and sourced from colorbrewer. The lighter the color, the less the number of deaths in the state and vice-versa.
  • The bubble map shows number of gun deaths in cities in the US. The size of the bubble corresponds with the number of deaths.
    The larger the bubble, the more the number of deaths and vice-versa.
  • The data has been normalized based on each states population to give an accurate representation of the statistics behind gun deaths.
    Users can interact with the viz by hovering around diff. states to get more info regarding number of deaths per gender and per age group.
  • Users can also manipulate data by filtering and grouping the data based on both age group and gender by selecting the corresponding values from the drop-down menu.
